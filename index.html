ПОЛИТИКА ОБРАБОТКИ ПЕРСОНАЛЬНЫХ ДАННЫХ

Дата вступления в силу: «24» апреля 2026 г.

Настоящая Политика регулирует порядок обработки персональных данных пользователей бота «Домовые чаты Москвы» (далее — Бот), функционирующего на платформе Max.

Оператор персональных данных

ОГРНИП: 324330000022254

email: ks.moskwa@yandex.ru

Оператор осуществляет обработку персональных данных в соответствии с Федеральным законом № 152-ФЗ «О персональных данных».

Какие данные обрабатываются

При использовании Бота могут обрабатываться следующие данные:

  • ID пользователя платформы Max
  • имя пользователя (никнейм)

Бот не запрашивает и не обрабатывает:

  • паспортные данные
  • номера телефонов
  • адреса
  • платежную информацию
  • специальные категории персональных данных

Цели обработки данных

Персональные данные обрабатываются исключительно в целях:

  • предоставления пользователю функционала бота
  • корректной технической работы бота

Правовые основания обработки

Обработка осуществляется на основании:

  • ст.6 ФЗ-152 — согласие субъекта персональных данных
  • пользовательского соглашения платформы Max
  • факта добровольного начала использования бота

Хранение и передача данных

Бот не осуществляет систематическое хранение персональных данных.

Данные используются исключительно в момент обработки запроса.

База персональных данных не формируется.

Передача третьим лицам не осуществляется.

Обработка происходит в облачной инфраструктуре на территории РФ.

Срок обработки данных

Поскольку хранение персональных данных не осуществляется, обработка носит разовый технический характер и прекращается после выполнения запроса.

Права пользователя

Пользователь имеет право:

  • получать информацию об обработке его данных
  • отозвать согласие путём прекращения использования бота
  • обращаться к Оператору по вопросам обработки данных

Меры защиты

Оператор принимает необходимые организационные и технические меры для защиты персональных данных в соответствии со ст. 19 Федеральный закон № 152-ФЗ «О персональных данных».
